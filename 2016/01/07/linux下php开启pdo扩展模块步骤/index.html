linux下php开启pdo扩展模块步骤

linux下php开启pdo扩展模块步骤

假设php目录为/usr/local/php
mysql目录为/usr/local/mysql

  1. 进入PHP源码包ext/pdo目录

cd ext/pdo

  1. 执行/usr/local/php/bin/phpize[假设PHP的安装目录为/usr/local/php]

/usr/local/php/bin/phpize

  1. 配置扩展pdo

./configure –with-php-config=/usr/local/php/bin/php-config –enable-pdo=shared

  1. 编译 && 安装pdo

make && make install
成功则出现
Installing shared extensions: /usr/local/php//lib/php/extensions/no-debug-non-zts-20060613/
Installing header files: /usr/local/php//include/php/
Installing PDO headers: /usr/local/php//include/php/ext/pdo/
说明在/usr/local/php//lib/php/extensions/no-debug-non-zts-20060613/目录下生成了pdo.so文件

  1. 在PHP源码包中进入pdo_mysql

cd ext/pdo_mysql

  1. /usr/local/php/bin/phpize
    /usr/local/php/bin/phpize
  2. 配置pdo_mysql[假设Mysql的安装目录为/usr/local/mysql]
    1
    ./configure –with-php-config=/usr/local/php/bin/php-config –with-pdo-mysql=/usr/local/mysql
  3. 编译 安装pdo_mysql
    make && make install
    成功则出现
    Installing shared extensions: /usr/local/php//lib/php/extensions/no-debug-non-zts-20060613/
    说明的/usr/local/php//lib/php/extensions/no-debug-non-zts-20060613/目录下生成了pdo_mysql.so文件
  4. 在php配置文件中加上

extension=”/usr/local/php//lib/php/extensions/no-debug-non-zts-20060613/pdo_mysql.so”;
extension=”/usr/local/php//lib/php/extensions/no-debug-non-zts-20060613/pdo.so”;

  1. 重启相关服务

/etc/init.d/php-fpm restart
